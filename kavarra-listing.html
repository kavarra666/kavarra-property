Unit Tersedia

Temukan unit terbaik di kawasan PIK2 — apartment, ruko, dan rumah untuk dijual maupun disewa.

📝 Panduan Tambah Unit Baru

Copy blok di bawah ini, paste setelah unit terakhir di array UNITS (sebelum tanda ];), lalu ganti semua teks sesuai unit baru. Jangan lupa tambahkan koma , setelah unit sebelumnya. 

  {
    id: 99,                          // ← Ganti angka, harus unik
    tipe: "Disewa",                  // ← "Dijual" atau "Disewa"
    judul: "Nama Unit",              // ← Judul listing
    lokasi: "Tower A Lt.5 No.502",  // ← Lokasi unit
    luas: "28 m²",                  // ← Luas unit
    kamar: "Studio",                // ← Studio / 1BR / 2BR / 3BR
    lantai: "Lt. 5",               // ← Lantai unit
    harga: "Rp 8.000.000",         // ← Harga jual atau sewa
    harga_sub: "/ bulan",          // ← "/ bulan", "/ tahun", atau "" jika jual
    deskripsi: "Deskripsi unit...", // ← Deskripsi singkat
    tiktok: "https://tiktok.com/@kavarraproperty", // ← Link video TikTok
    foto: [                         // ← Link foto dari Google Drive
      "https://drive.google.com/uc?export=view&id=FILE_ID_1",
      "https://drive.google.com/uc?export=view&id=FILE_ID_2",
    ]
  },

Cara dapat link foto dari Google Drive:
1. Upload foto ke Google Drive
2. Klik kanan foto → Share → Anyone with the link → Copy link
3. Link format: https://drive.google.com/file/d/FILE_ID/view
4. Ambil FILE_ID → paste ke template di atas: ...export=view&id=FILE_ID
